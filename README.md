# BoatAPI
BoatAPI is a REST API that allows for CRUD operations for the Boat Application.

## Usage
### Swimlanes
| Operation  | Request | URL |
| ------------- | ------------- | ------------- |
| Read  | GET  | /swimlane |
| Create  | POST  | /swimlane |
| Read  | GET  | /swimlane/{id} |
| Update | PUT | /swimlane |
| Delete | DELETE | /swimlane/{id} |
### Boats
| Operation  | Request | URL |
| ------------- | ------------- | ------------- |
| Read  | GET  | /boat |
| Create  | POST  | /boat |
| Read  | GET  | /boat/{id} |
| Update | PUT | /boat |
| Delete | DELETE | /boat/{id} |
